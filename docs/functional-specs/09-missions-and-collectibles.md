# 09 - Missions and Collectibles (Phase 2)

## Missions

### Overview
This section allows viewing and editing available and active missions (contracts) in the game.

### Mission List
Each mission is displayed with:
- **Mission type** (harvest, plowing, cultivation, weeding, wood transport, deadwood)
- **Status** (available, in progress, completed)
- **Reward** in money
- **Related field** (for farming missions)
- **Crop type** (for harvest missions)
- **Progress** (completion percentage for active missions)

### Simple Mode
- **Reward**: edit the mission reward amount
- **Progress**: edit the completion percentage
- **"Complete mission" action**: sets completion to 100% and status to completed

### Advanced Mode
- **Status**: manually change the mission status
- **Reimbursement**: edit the reimbursement amount
- **End date**: edit the mission deadline
- **Type-specific parameters**: expected/deposited liters (harvest), number of trees (transport), target weed state (weeding)

## Collectibles

### Overview
The game contains 25 collectible items hidden across the map. This section allows tracking and editing their state.

### Display
- List of 25 collectibles with their number (1-25)
- Status: collected or not collected
- Counter: X/25 collected

### Editing
- Toggle each collectible's state individually
- **"Collect all" action**: mark all collectibles as collected
- **"Reset all" action**: reset all collectibles to not collected

## Contract Settings (Advanced Mode)

Edit the contract system limits:
- **Leased vehicles**: maximum number of simultaneously leased vehicles
- **Missions per farm**: maximum number of active missions per farm
- **Clearing limit**: clearing addition limit
